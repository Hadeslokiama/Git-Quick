## Overview

This guide provides a complete, beginner-friendly setup for Git, covering installation, essential configuration, SSH key setup, and a compact cheat sheet of commonly used commands. It also introduces a global commit convention to ensure consistent, clear, and well-documented commit messages.

Whether you are starting a new project or collaborating on existing repositories, this guide helps you:

- Install Git across different operating systems (Windows, macOS, Linux).
- Configure your identity globally for proper commit attribution.
- Securely connect to GitHub using SSH keys.
- Quickly reference essential Git commands for daily workflow.
- Follow a standard commit message format to maintain a clean and understandable history.

This initiative aims to make Git approachable for newcomers while promoting best practices for collaboration and version control.
